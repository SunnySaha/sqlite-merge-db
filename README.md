# SQLite DB Merger (Windows)

A simple and lightweight Windows desktop tool to merge two SQLite database files
into a single database using a graphical user interface.

The tool automatically ignores duplicate records and does not require Python
to be installed on the system.

---

## Features
- GUI-based (Tkinter)
- Merge two SQLite databases into one
- Duplicate records automatically ignored
- Creates merged database at user-selected location
- No Python installation required
- Works on Windows 10 / 11 (64-bit)

---

## How to Use
1. Download the EXE from the Releases section
2. Run `sqlite-db-merger.exe`
3. Select SQLite DB file 1
4. Select SQLite DB file 2
5. Choose output location for merged DB
6. Click **Merge Databases**
